# 📊 청년층 조기퇴직 요인 분석 프로젝트

본 프로젝트는 청년층의 조기퇴직 현상에 영향을 미치는 요인을
데이터 기반으로 분석하는 팀 프로젝트입니다.

---

## 📌 프로젝트 개요
- **주제**: 청년층 조기퇴직에 영향을 미치는 요인 분석
- **목표**
  - 연도별 취업·퇴직 추이 분석
  - 근속기간, 임금, 전공 일치 여부 등 주요 변수 분석
  - 조기퇴직과의 관계를 데이터로 설명

---

## 🧑‍🤝‍🧑 팀 구성 및 브랜치 운영
본 프로젝트는 **5명의 팀원이 각자 개별 브랜치에서 작업**하는 방식으로 진행되었습니다.

| 팀원 | 브랜치 |
|---|---|
| BR | team2/BR |
| ES | team2/ES |
| GY | team2/GY |
| MJ | team2/MJ |
| SY | team2/SY |

> `main` 브랜치는 최종 결과 정리 및 README 용도로 사용됩니다.  
> 실제 분석 코드는 각 팀원 브랜치에서 확인할 수 있습니다.

---

## 📂 브랜치별 역할 (예시)
- **team2/BR** : 연도별 취업·퇴직자 수 분석
- **team2/ES** : 근속기간 및 단기근속 비율 분석
- **team2/GY** : 지역별 조기퇴직 비교
- **team2/MJ** : 임금 수준과 퇴직률 관계 분석
- **team2/SY** : 전공 일치 여부 및 근로환경 만족도 분석

---

## 🛠 사용 기술
- Python

- matplotlib / seaborn

## 📈 주요 분석 내용
- 연도별 취업자·퇴직자 추이
- 단기 근속(1년 미만) 비율 분석
- 임금 수준과 조기퇴직의 관계
- 전공 일치 여부와 퇴직률 비교
- 근로환경 만족도 분석
